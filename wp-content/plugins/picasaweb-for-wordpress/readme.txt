== Description ==

Easy to insert in the pages / posts PicasaWeb Albums.Based on jQuery.  
If you have any question or find any bug, you can visit [Plugin Homepage](http://immmmm.com/picasaweb-for-wordpress.html "PicasaWeb for WordPress") to submit it.

基于jQuery。方便在文章或页面中插入Picasa网络相册。

== Installation ==

It's very easy to install the plugin.

1. Upload the whole directory to "/wp-content/plugins/" directory
2. Activate the plugin through the "Plugins" menu in WordPress.
3. That's all.
